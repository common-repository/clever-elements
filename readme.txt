== Description ==

= Clever Elements email marketing for WordPress =

This plugin utilizes a third-party service, Clever Elements, for certain functionalities related to email marketing. Clever Elements operates under the domains cleverelements.com and sendcockpit.com.

* This plugin allows you to easily connect your Wordpress to your Clever Elements account
* Extend your existing forms with an option for newsletter sign up.
* Generate a form for newsletter sign ups and integrate the HTML code anywhere.
* This is the official Clever Elements WordPress plugin. If you have questions or need help, please do not hesitate to contact us: http://support.cleverelements.com/ or support@cleverelements.com.

Please review Clever Elements' terms of use and privacy policy for more information:
- Terms of Use: [https://www.cleverelements.com/en/agb/](https://www.cleverelements.com/en/agb/)
- Privacy Policy: [https://www.cleverelements.com/en/datenschutz/](https://www.cleverelements.com/en/datenschutz/)

== Installation ==
1. Just go to "Plugins" -> "Add New" and search for "Clever Elements"
1. Or download the plugin from https://wordpress.org/plugins/clever-elements/
1. **Check if php SOAP module is enabled on your server**
1. Activate the plugin
1. Create a Clever Elements account (http://www.cleverelements.com/_lnd/idaf:38845) and connect the Clever Elements plugin to your account via ID and api key (http://goo.gl/oLxYpc)
1. Enjoy and contact us, if you have any questions: support@cleverelements.com

== Screenshots ==
1. Connect to Clever Elements API using the credentials provided after having logged in into the system. If everything is fine you will see the message "Connected".
2. Choose from your subscriber lists, set double opt-in if necessary, set where you want subscribers to subscribe from or change checkbox label.
3. Form setup. Choose from your subscriber lists, set double opt-in if necessary, change message that subscriber should see if he subscribes or any error occurs during the subscription.
